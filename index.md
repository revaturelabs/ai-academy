# AI Academy

Welcome to the **AI Academy** curriculum. This site publishes two complete,
self-paced learning tracks. Every topic is a reading with worked examples,
diagrams, and its own **Previous / Table of Contents / Next** navigation, so you
can move through a track start to finish or jump straight to a topic.

---

## 📘 CIT — AI-Native Foundations

Computational thinking, AI systems, applied maths for AI, human oversight, and
Python for AI (PE, VC & RAG). **15 weeks · 5 modules.**

➡️ **[Open the CIT curriculum index](CIT/)**

- **M1 — Computational Thinking** — how machines think, specifying for AI
- **M2 — Introduction to AI Systems** — LLMs, the 2026 AI stack, ethics & governance
- **M3 — Applied Maths for AI** — vectors, probability, model evaluation
- **M4 — Human Cognition & AI Oversight** — judgment, bias, human-in-the-loop
- **M5 — Python for AI, PE, VC & RAG** — Python foundations through RAG & agents

---

## 🐍 Crescent — AI-Native Engineering Foundations

A hands-on Python engineering track, with Part B sharing the AI foundations
modules (M1–M3) drawn from the CIT tree.

➡️ **[Open the Crescent curriculum index](Cresent/Technical/)**

- **P1–P6** — Python syntax, control flow, data structures, classes, files &
  exceptions, and Git / portfolio
- **Part B (M1–M3)** — the shared AI-native foundations content

---

### How to use this site

- Start from a track index above and pick a topic.
- Inside every reading, use the **Prev / TOC / Next** links to move through the
  sequence in delivery order.
- Diagrams are embedded inline; quizzes accompany many topics.
