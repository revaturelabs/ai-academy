# AI Native Engineering Foundations

Welcome. This site publishes the **Crescent Technical** learning track: a
hands-on Python engineering course, followed by shared AI-native foundations.
Every topic is a reading with worked examples and diagrams, and each page has
its own **Previous / Table of Contents / Next** navigation so you can move
through the track in order or jump straight to a topic.

➡️ **[Open the full curriculum index](Cresent/Technical/)**

---

## What's inside

### Part A — Python Engineering

| Module | Focus |
|---|---|
| **P1 — Python Foundations & Syntax** | The environment, variables & types, operators, statements |
| **P2 — Control Structures, Functions & Tooling** | Conditionals, loops, functions, modules & packaging |
| **P3 — Data Structures** | Lists, tuples, sets, dictionaries, iterators & generators |
| **P4 — Classes & Objects** | OO foundations, inheritance, special methods & dataclasses |
| **P5 — Files & Exception Handling** | File handling, errors & exceptions, a robust-reader case study |
| **P6 — Git, GitHub & Portfolio** | Version-control basics and portfolio diagnostics |

### Part B — AI Foundations (shared with CIT)

| Module | Focus |
|---|---|
| **M1 — Computational Thinking** | What computation is, expressing logic, specifying for AI |
| **M2 — Introduction to AI Systems** | LLMs, the 2026 AI stack, how models behave, ethics & governance |
| **M3 — Human Cognition & AI Oversight** | Human judgment, bias, and human-in-the-loop case studies |
